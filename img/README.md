# Galerie fotek - Hokejbal

Sem vlož fotografie ze svých hokejbalových zápasů a akcí.

## Jak přidat fotografie:

1. Stáhni si fotografie z těchto webů:
   - https://www.hokejbal.cz/hrac?id=49963
   - https://cechyzapad.cmshb.cz/hrac?id=49963
   - https://rebelpraha.cz/match/84
   - Instagram: https://www.instagram.com/p/DQqvk5wgE9y/
   - Facebook: https://www.facebook.com/skrebelpraha/

2. Ulož je do této složky (`img/`)

3. Pojmenuj je takto:
   - `hockey-1.jpg`
   - `hockey-2.jpg`
   - `hockey-3.jpg`
   - `hockey-4.jpg`

4. Fotografie se automaticky zobrazí v galerii na webové stránce!

## Formáty:
- Podporované: JPG, PNG, WEBP
- Doporučená velikost: cca 600x600px
- Všechny fotky se zobrazí jako čtverec (1:1)
